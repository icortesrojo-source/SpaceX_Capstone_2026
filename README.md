## SpaceX Falcon 9 Landing Prediction

Final Capstone Project for the IBM Data Science Professional Certificate.

## Objective

Predict whether the Falcon 9 first stage will land successfully using publicly available launch data.

## Project Workflow

- Data Collection (REST API & Web Scraping)
- Data Wrangling
- Exploratory Data Analysis
- SQL Analysis
- Interactive Visual Analytics (Folium & Plotly Dash)
- Predictive Modeling

## Technologies

- Python
- Pandas
- BeautifulSoup
- SQL
- Folium
- Plotly Dash
- Scikit-learn
